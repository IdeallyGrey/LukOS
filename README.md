# LukOS
A basic x86 assembly kernel written in 12 hours at Dickinson's S25 Hackathon.

**Goals:**
*Realistic*
[ ] Boot
[ ] Print text via BIOS
[ ] Protected mode 
[ ] Exceptions and interupts (timer, keyboard)

*Wishful thinking*
[ ] Memory management

*Pipe dream*
[ ] Premptive multitasking
[ ] Shell
